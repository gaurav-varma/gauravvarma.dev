Gaurav Varma Website and Blog.

## Built with

- Node.js v20.15.0
- React.js v17.0.2
- Next.js v12.1.0

## Getting Started

- Install dependencies with `yarn` or `npm`
- Start the dev server with `yarn dev` or `npm run dev`
- Now you can see the app running here [http://localhost:3000/](http://localhost:3000/)
